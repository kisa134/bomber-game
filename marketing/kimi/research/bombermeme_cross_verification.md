  - **Combo Montages:** 20-30 сек (TikTok/Shorts — speed ramping)
  - **Tutorial Shorts:** 30-45 сек (YouTube Shorts — how-to)
  - **Story Clips:** 45-60 сек (YouTube Shorts — narrative arc)
- Консенсус: короче = лучше для viral potential; длиннее = лучше для storytelling/tutorials
- YouTube Shorts sweet spot 30-60 сек (консенсус Dim03+Dim06); TikTok 15-30 сек (консенсус Dim03+Dim05); Reels 7-30 сек (консенсус Dim03+Dim07)

### Конфликт 2: Text-first vs Video-first на X (Twitter)

**Проблема:** Противоречивые claims о том, какой формат лучше на X.

- **Dim04 Claim A (PostEverywhere/Buffer):** Text-only получает на 30% больше engagement, чем видео. X — единственная major платформа, где текст превосходит видео [^11^].
- **Dim04 Claim B (AdLibrary/Graham Mann):** Native video получает в 10x больше engagement, чем text-only [^12^] [^128^].
- **Dim04 Claim C:** 4 из 5 пользовательских сессий на X включают просмотр видео [^22^].

**Разрешение:**
- **Оба claims верны в разных контекстах.** Text-first = лучше для reply/conversation engagement. Video-first = лучше для impressions/reach (алгоритм бустит video для конкуренции с TikTok).
- **Рекомендация для Bombermeme:** Использовать text для conversation-driven постов (tips, hot takes, polls); video для reach/awareness (gameplay clips). Комбинировать: text-post с нативным видео-embed. Dim04 раздел "Controversies" предоставляет эту резолюцию явно.

### Конфликт 6: Canvas 2D vs WebGL для игрового движка

**Проблема:** Оценка технической архитектуры diverges между измерениями.

- **Dim01:** Canvas 2D = adequate для MVP, hard ceiling ~10K objects at 60fps [^144^], но не future-proof
- **Dim01:** Single-threaded Node.js = hard ceiling ~2,000 concurrent players [^212^]
- **Dim09:** WebGL/Three.js не упоминается; фокус на AI content pipeline
- **Dim10:** Sonic SVM gaming L2 предлагает dedicated performance [^371^]

**Разрешение:**
- Canvas 2D приемлем для MVP (Bomberman = grid-based 2D, не требует WebGL). Однако scaling ceiling в 2,000 players — более критичный bottleneck, чем renderer. Приоритет: horizontal sharding > renderer upgrade. WebGL migration — Phase 2 (post-10K DAU).

### Конфликт 7: TikTok crypto ads — условно разрешены или запрещены?

**Проблема:** Политика TikTok по крипто-рекламе описана по-разному.

- **Dim05 (TikTok Branded Content Policy):** Financial services branded content запрещён [^71^]
- **Dim03:** Не упоминает политику; фокус на organic content

**Разрешение:**
- **Organic + UGC strategy = primary path.** Spark Ads (буст organic posts) возможен только после platform approval. Прямая крипто-реклама (ICO/DeFi) = полностью запрещена. Gaming content без упоминания токена = разрешён. **Bombermeme должен фокусироваться на organic UGC + creator-first strategy, не paid TikTok ads.**

### Конфликт 8: Pixels DAU — 1.2M vs реальный retention слабее
