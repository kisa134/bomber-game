## Сквозные пассы (применяются ко ВСЕМ блокам)
- **A. Копи с отношением** — заменить объясняющий маркетинг на короткие удары/провокации (`WIN OR GET WIPED`, `THE CHAIN DOESN'T LIE`, `LAST MEME STANDING`).
- **B. Словарь переходов** — каждый стык = Character Run / Explosion Wipe / Scanline Dissolve. Никаких жёстких границ.
- **C. 3 плана глубины** — bg-атмосфера (sticky/parallax) · mid-контент · fg-breakout (overflow за контейнер).
- **D. Asset-batch** — сгенерить через `tools/asset-gen`: env-плиты, fg-силуэты/дебрис, explosion/smoke, loop-видео-плиты. Слоты уже в коде §3.
- **E. Mobile / reduced-motion** — pin/параллакс off <768px, вертикальный нарратив, атмосфера сохранена.
- **F. Перф/a11y** — lazy video/Rive (IntersectionObserver), ScrollTrigger `invalidateOnRefresh`, reduced-motion фоллбэки.
