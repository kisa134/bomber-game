## 0. МИРОВОЗЗРЕНИЕ

Лендинг — не набор блоков, а **управляемое путешествие сквозь вселенную игры**. Скролл = монтаж: каждый пиксель прокрутки — следующий кадр нарратива. Пользователь не «читает о продукте» — он **присутствует на арене**, понимает лор через среду, а не через текст.

**Главный принцип:** один экран — одна эмоциональная задача. **Напряжение → Раскрытие → Вовлечение → Решение.** Сцена сначала создаёт вопрос, потом отвечает визуалом, а не параграфом.

**Правило арт-дирекшна — ОТНОШЕНИЕ (не вежливость):** лендинг не должен ощущаться вежливым или generic. У каждой секции — **свой эмоциональный удар и чёткий характер**, при этом все принадлежат **одному непрерывному кино-миру**. Никакой пассивной подачи. Каждый блок компонуется как **заявление, провокация или геймплейный бит напряжения** — не как объясняющий маркетинговый лейаут. Предпочтение: **короткий копирайт, дерзкое кадрирование, уверенный визуальный ритм** вместо разжёвывающих текстов. Дерзко, но в рамках общей режиссуры и чёрной дисциплины.

### 5 эстетических слоёв (всегда работают ОДНОВРЕМЕННО)
1. **Pixel Art / Retro Futurism** — осознанная аркадная ностальгия + строгая геометрия. Пиксель = акцент/текстура (HUD-шрифты, спрайт-анимации через `steps()`, тайл-сетка), не весь интерфейс.
2. **Glassmorphism** — полупрозрачность ТОЛЬКО над богатым живым фоном. `backdrop-filter: blur(20px)` + `oklch(1 0 0 / 0.06)`.
3. **Broadcast / Arena HUD (diegetic UI)** — интерфейс ВНУТРИ мира: таймеры, счёт, LIVE, scan-линии, трансляции как часть арены.
4. **Glitch / CRT** — артефакт как СОБЫТИЕ, не фон. Chromatic aberration на переходах, scanlines на экранах, clip-path glitch в момент взрыва.
5. **Meme Culture Archetype** — персонажи = архетипы: Трамп = Chaos Agent, Маск = Tech Overlord. Арена = гладиаторский театр из мемов.

### Z-архитектура (глубина пространства)
```
z-0   Задник: туман, частицы, матрица, арена вдали (медленно, тёмно, blur)
z-10  Средний план: арты, экраны, объекты сцены
z-20  Стекло: HUD, карточки, info-панели (glassmorphism)
z-30  Передний план: персонажи, взрывы, частицы (overflow:visible, выступают за контейнер)
z-40  Постоянный UI: nav, sticky
z-50  Оверлеи: модалки
```
Персонажи на границах секций: `position:absolute; z-index:30; overflow:visible` на родителе — физически выступают за контейнер.

### Цвета (OKLCH, семантические роли = источники света)
```css
--color-void:       oklch(0.08 0.025 250);  /* почти-чёрный с синим — ночное небо арены, НЕ #000 */
--color-surface:    oklch(0.12 0.02 250);
--color-glass:      oklch(1 0 0 / 0.05);
--color-explosion:  oklch(0.85 0.28 60);    /* взрыв/огонь */
--color-neon:       oklch(0.75 0.30 290);   /* фиолетовый неон */
--color-live:       oklch(0.65 0.35 140);   /* зелёный LIVE */
--color-gold:       oklch(0.82 0.22 85);    /* золото — победа/топ */
--color-text:       oklch(0.95 0.01 100);
--color-text-muted: oklch(0.65 0.01 100);
```
> Примечание: текущий код использует gold `#f5c842`, teal `#3a9e9e`, red `#d44030`. При внедрении OKLCH держать в lockstep, не плодить два набора.

### Техническая основа
- **GSAP + ScrollTrigger** — вся скролл-режиссура (pinned, scrub, stagger)
- **Lenis** — инерционный smooth scroll, sync с ScrollTrigger: `lenis.on('scroll', ScrollTrigger.update)` + `gsap.ticker.add(t => lenis.raf(t*1000))`
- **Rive** — интерактивные 2D-персонажи со state machines (idle/run/victory)
- **CSS Scroll-Driven** (`animation-timeline: view()`) — простые reveal (карточки/текст), off-main-thread
- **CSS Variable Fonts** — кинетика через `font-variation-settings`
- **Canvas / Three.js** — частицы взрыва, матрица, ambient

## ГЛОБАЛЬНЫЕ ПЕРЕХОДЫ (смысловые швы, один словарь на весь сайт)
- **Тип A — Character Run Bridge:** персонажи «добегают» из секции в секцию. CSS `steps()` спрайт, `overflow:visible`. **Фирменный мотив, повторяется 3–4 раза.**
- **Тип B — Explosion Wipe:** вспышка из центра `radial-gradient` + `scale:0→1` + clip-path reveal. Самый сильный — **только 1–2 раза на сайт** (дешевеет при злоупотреблении).
- **Тип C — Scanline Dissolve:** CRT-распад через scanlines + chromatic aberration. Мягкий, технический, для HUD→инфо переходов.

## СЕКЦИЯ 1 — HERO
**Задача:** за 3 сек — «это мем-арена, стильно, живая вселенная», не «что за игра», а «хочу быть внутри».
- **Лево (60%):** заголовок пиксельным display (`Press Start 2P`/`VT323`), буквы приходят по одной `stagger:0.05`; суб 1-2 слова («MEME BATTLE ARENA»); CTA стеклянная, hover=explosion glow; 3 micro-stat (онлайн/матчей/арен) — живые данные.
- **Право (40%) — МЕДИА-СЦЕНА = окно в мир:** broadcast-frame со стеклянной рамкой (z-20), внутри loop-видео/animated art арены (z-10), HUD-overlay: верх «ARENA BROADCAST» + мигающий зелёный LIVE, низ — счётчик зрителей + таймер матча; sci-fi пиксельные уголки рамки.
- **Фон:** gradient void→теплее книзу; дрейфующие частицы; тонкий scanline `repeating-linear-gradient` 1px `opacity:0.03`.
- **Анимация входа (GSAP):** `.char` stagger x:-20 → subtitle y:10 → cta scale:0.9 → media `clipPath: inset(0 100% 0 0)` → stats stagger.
- **Переход:** Тип A — два спрайта убегают вправо за нижнюю границу.

## СЕКЦИЯ 2 — CURTAIN REVEAL (Шторки)
**Задача:** самый театральный момент — напряжение → взрыв → раскрытие арены. «ЭТО началось».
**Pinned section, ~250vh, scrub:0.4**, 4 акта:
- **Акт 1 (0–20%) PRE-SHOW:** шторки закрыты, фон void; **Matrix на самих шторках** (Canvas `fillText` пиксельным, `oklch(0.75 0.30 290/0.7)` — падающие бомбы/смайлы/лого); Трамп (лево, профиль вправо), Маск (право, профиль влево).
- **Акт 2 (20–55%) THE RUN:** бегут навстречу, `animation: run steps(8)`; `translateX` по scrub: Трамп 0→-35%, Маск 0→+35%; матрица усиливается.
- **Акт 3 (55–80%) THE OPENING:** раздвиг `clipPath`/`translateX` ±100%; персонажи уезжают со шторками; пробивается свет `radial-gradient(circle at 50% 50%, oklch(0.85 0.28 60/0.4), transparent 60%)`; light burst opacity 0→0.8→0.3.
- **Акт 4 (80–100%) THE EXPLOSION:** центр — взрыв (Canvas pixel-партиклы оранж/жёлтые); glitch 0.3с (`hue-rotate(180)` + chromatic aberration через псевдоэлементы); за взрывом — первый кадр Story.
```css
.curtain::after{content:'';position:absolute;inset:0;pointer-events:none;
 background:repeating-linear-gradient(0deg,oklch(0 0 0/0.15) 0,transparent 1px,transparent 3px);}
```

## ПЕРФОРМАНС
- `<img>/<video>`: `loading=lazy`, `decoding=async`, заданы `width/height`.
- Rive: lazy через `IntersectionObserver`. Canvas: pause rAF вне viewport.
- ScrollTrigger: `invalidateOnRefresh:true`. Lenis+GSAP sync (см. выше).

## ДОСТУПНОСТЬ
- `aria-label`, Tab-достижимость.
- `@media (prefers-reduced-motion: reduce)` → GSAP `duration:0.01`, transitions off, Rive пауза на 1-м кадре.
- Видео `autoplay muted loop playsinline` + `<track>` для трейлера.
- Контраст текст `oklch(0.95)`/фон `oklch(0.08)` ≈ 18:1 (AAA).

## МИКРОИНТЕРАКЦИИ
- **Курсор (desktop):** пиксельный прицел 32×32 (`cursor: url(crosshair.cur), crosshair`); hover на кликабельное → bomb sprite.
- **Звук (opt-in toggle в header):** 8-bit hover/click, fanfare на победу, boom на взрыв; `<audio>` volume 0.3 / WebAudio.
- **Hover персонажа:** Rive `idle→attention` (смотрит на курсор, wave).
- **Пасхалки:** Konami (`↑↑↓↓←→←→BA`) → rainMode (персонажи падают); 5 кликов по лого → 15с pixel-cutscene «История арены»; AFK 60с → персонаж выбегает и тыкает.
