# Sounds

Drop audio files here with the **exact names** below. Missing files are simply
silent — nothing breaks. Music/SFX can be toggled in the in-game Settings.

- Format: **MP3** (`.mp3`). (Want `.ogg`? tell me and I'll switch the paths.)
- SFX: short (0.1–0.6 s), **< 20 KB** each.
- Music: seamless loops, keep **< ~1 MB** each.
- Audio only starts after the first tap/click (browser autoplay rules).

## SFX (all wired)

| File | Plays when |
| --- | --- |

| `explode.mp3` | a bomb explodes |
