# Sounds

Drop audio files here with the **exact names** below. Missing files are simply
silent — nothing breaks.

- Format: **MP3** (`.mp3`) — widest browser support. (If you prefer `.ogg`,
  tell me and I'll switch the paths.)
- Keep each file **< 20 KB** and short (0.1–0.5 s for SFX).
- Audio only starts after the first click (Quickplay), per browser autoplay rules.

| File | Plays when |
| --- | --- |
| `place.mp3` | You drop a bomb |
| `explode.mp3` | A bomb explodes |
| `pickup.mp3` | A powerup is collected |
| `death.mp3` | A player dies |
