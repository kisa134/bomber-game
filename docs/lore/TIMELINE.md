# Хронология мира BomberMeme World

> *«История — это просто цепочка взрывов. Одни разрушают. Другие создают. Узнать разницу — вот искусство.»*
> — Хроникёр Скribe, Решётчатый Синдикат

## Эпоха Создания (До Года 0)

| Год | Событие |
|-----|---------|
| **∞** | Существует лишь Первичная Пустота. Великий Протон спит в центре небытия. |
| **-10 000** | Первое Деление. Великий Протон раскалывается на Семь Искр. Каждая устремляется в противоположный угол пустоты, оставляя след бомбической энергии. |
| **-5 000** | Рождение Прото-Мира. Семь Искр встречаются в центре и создают первую твердую землю. Великий Протон становится Сердцевиной мира. |
| **-1 000** | Появление первых Блобов — бесформенных существ из чистой энергии. |
| **-500** | Эволюция Блобов в Прото-Героев. Первые мемные формы: лягушки, собаки, роботы, черепа. |
| **-100** | Прото-Герои обнаруживают первые естественные Бомбы — кристаллы конденсированной бомбической энергии. |
| **-10** | Сердцевина достигает критической массы. Предвестники Великого Взрыва: землетрясения, вспышки энергии, видения будущего. |
